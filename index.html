História dos computadores

O primeiro computador surgiu em 1944 conhecido como MARK I, sua operação era feita de forma manual, projetado para realizar diferentes tipos de cálculos, o computador era tão grande, que ocupava uma sala inteira.

Com o passar dos anos a tecnologia foi evoluindo, melhorando a capacidade dos computadores, reduzindo seu tamanho, até chegar ao que conhecemos hoje, a 4º geração, mas há quem diga que já estamos vivenciando a 5º geração.

O que são Clients?

De acordo com a linguagem da internet, clientes são os dispositivos conectados a uma rede, onde fazem uma solicitação e tem uma resposta.

Uma pessoa ao acessar o navegador Web, digita um site ou faz uma busca, a partir desse momento o equipamento utilizado "CLIENTE", terá um retorno da pagina "SERVER".

O que são Servers?

Server’s são responsáveis por receberem dados enviados pelos clientes, armazenam, tratam e retornam com a resposta novamente para os clientes.

Entendendo a diferença entre listas ordenadas e não ordenadas

Listas ordenas: como o próprio nome já diz, são listas que contem uma ordem e são enumeradas. Ex:

  • Item 1
  • Item 2
  • Item 3
    •

Listas não ordenadas: são lista que não possuem uma ordem, mas no início de cada item possui uma marcação. Ex:

  1. Item 1
  2. Item 2
  3. Item 3
    4.

Como é uma estrutura básica de um HTML

<!DOCTYPE html>
 Tag repsonsável por dizer ao navegador que o arquivo é um documento do tipo HTML.

<html lang="pt-br">
 Tag que abrange todo o conteudo html, incia no top código com html e termina com /html, desta forma marcando todo conteúdo. Dentro dessa mesma tag também está informando em qual idioma deverá ser exibido.

<head>
 Tag responsável pela cabeça do site, envolve o titulo da página, estilos e outros arquivos.

<meta charset="UTF-8">
 Tag de configuração, aqui dizemos ao navegador qual é o formato de codificação deve ser utilizado.

meta http-equiv="X-UA-Compatible" content="IE=edge">
 Tag de compatibilidade com internet explorer

<meta name="viewport" content="width=device-width, initial-scale=1.0">
 Tag responsavel por controlar a exibição do nosso site em dispositivos móveis.

<title> Document </title>
 Tag usada para informar o titlo do site, aparece na aba do navegador.

</head>
 Tag de fechamento da cabeça do nosso html.

<< /span>body>
 Tag que contem os códgigos escritos, os quais serão exibidos na pagina de criação,podemos inserir: paragrafos, listas, imagens, mídia, etc.

</body>
 Tag de fechamento do corpo.

</html>
 Tag de fechamento do documento html.

Apresentando algumas Tag's aprendidas

  • <a> - Tag utilizada para definir um hiperlink. (Ancora)

  • <abbr> - Tag é utilizada para marcar uma sigla como HTML. ela adiciona a descrição via propriedade title.

  • <b> - Tag é utilizada para marcar um conteúdo como negrito.

  • <blockquote> - Tag é utilizada para criar citações de outra fonte no texto.

  • <strong> - Tag semelhante ao b sua diferença está no valor semântico, pois ela é responsável por dar uma forte ênfase no texto, fundamental para leitores de tela e acessibilidade.

  • <del> - Tag utilizada na correção de textos: exemplo.

  • <em> - Tag utilizada para dar uma enfase no conteudo, deixando o texto em itálico.

  • <font> - Tag foi usada no HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto. Essa tag foi descontinuada e não é suportada no HTML 5.

  • <i> - Está tag é utilizada para deixar o conteudo em italico.

  • <li> - Define um item da lista

  • <mark> - Tag utilizada para marcações no texto Exemplo.

  • <ol> - Tag utilizada para criar uma lista ordernada.

  • <p> - Tag utilizada para escrever um paragrafo.

  • <sub> - Tag coloca um conteudo a baixo da linha, ex: h2o

  • <small> - Tag define um texto menor

  • <sup> - Tag coloca um conteudo acima da linha, ex: 22

  • <u> - Tag sublinha um texto underline

  • <ul> - Tag utilizada para criar uma lista não ordenada.
